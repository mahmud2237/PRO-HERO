Discuss the scope of var, let, and const

Basically there are three kinds of scopes. Global, Local & Block Scope .Here var can have a global or local scope. When it declared outside function it is global scope, weather it declared inside function it is a local scope.Here let can have a global or local or block scope. When it declared in a block then it can be block scope.We can't access these variables outside the block. Another variables declared by const are similar to let variables.Such variables can have global, block or local scope.

Tell us the use cases of null and undefined

null & undefined both values are represent the absence of a value.But their use aren't same. We use these values in different conditions.So, if I assign the value null in a variable that means the variable has no value. We use null when we don't have any value of a variable or a object. And if we declared a variable but didn't assign any value than we get it's value as undefined. Also if we don't pass any parameter to a function and return nothing then after calling the function we get undefined. Basically it is a default value where any variable or object is not initialized.

What do you mean by REST API?
REST API means Representational State Transfer API ,referred to as RESTful API.It is a set of architectural constraints and principles used for designing networked applications. It’s sometimes referred to as a contract between an information provider and an information user—establishing the content required from the consumer and the content required by the producer. For example, the API design for a bus service could specify that the user supply a time or information and that the producer reply with specific answer, as which buses are available, ticket price and travel time etc.
